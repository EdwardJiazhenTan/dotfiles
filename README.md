my settings

Neovim: A highly configurable text editor built on Vim's foundations, enabling efficient text editing. The lazy.vim configuration aims to simplify the setup and management of Neovim plugins and settings.

iTerm2: A replacement for the default Terminal app in macOS, offering features like split panes, search, and customizations that enhance the command-line experience.

LinearMouse: A utility for macOS that disables mouse acceleration, particularly useful for external mice, to provide a linear and consistent pointer movement that's often preferred for precision tasks.

RStudio: An IDE for R

Yabai: A window management utility for macOS that allows for advanced window tiling, management, and automation, inspired by Linux window managers. It offers features like automatic layout adjustments and workspace management.

skhd (Simple HotKey Daemon): A simple and fast hotkey daemon for macOS with configurable keybindings, often used in conjunction with Yabai to create keyboard shortcuts for window management tasks.

SketchyBar: A highly customizable status bar for macOS, designed to replace or enhance the default system bar. It allows for extensive customization to display various system stats, control elements, and more, tailored to individual preferences.
